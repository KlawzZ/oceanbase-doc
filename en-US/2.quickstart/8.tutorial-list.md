# Quick start tutorials

You can watch tutorial videos to quickly learn about the installation, deployment, and basic operations of OceanBase Database.

This topic provides tutorials on how to quickly deploy an experience environment of OceanBase Database and how to quickly build applications and core features of OceanBase Database.

This topic provides the following tutorials:

* Deploy an experience environment of OceanBase Database

   * [Use OBD to quickly deploy OceanBase Database](# Use %20OBD%20 to quickly deploy %20OceanBase%20 Database)

   * [Use Docker to quickly deploy OceanBase Database](# Use %20Docker%20 to quickly deploy %20OceanBase%20 Database)

* [Basic operations of OceanBase Database](# Basic operations of OceanBase%20 Database)

* Create sample applications

   * [Create a Python application](# Create a %20Python%20 application)

   * [Create a C application](# Create a %20C%20 application)

   * [Create a Go application](# Create a %20Go%20 application)

   * [Create a Java application](# Create a %20Java%20 application)

* Experience advanced features of OceanBase Database (coming soon)

   * Run the TPC-C benchmark in OceanBase Database

   * Experience the hot row update capability of OceanBase Database

   * Experience operational OLAP

   * Experience parallel import and data compression

   * Experience the multitenancy feature

   * Experience DDL
