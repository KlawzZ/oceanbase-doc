enable_global_freeze_trigger 
=================================================

enable_global_freeze_trigger 用于设置是否开启自动触发全局冻结。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | </li><li> True   </li><li> False    |
| 是否重启 OBServer 节点生效 | 否                                                                                                      |

<main id="notice" type='explain'>
  <h4>说明</h4>
  <ul>
  <li> 该配置项从 V4.0.0 版本开始弃用。  </li>
  <li> 该配置项保持保持默认值即可，不推荐使用。 </li>
  </ul>
</main>
