# ob_sql_plan_memory_percentage

`ob_sql_plan_memory_percentage` 用于设置租户 sql_plan 值的内存百分比。

| **属性**      |      **描述**                            |
|---------------|------------------------------------------|
| 参数类型      | int                                       |
| 默认值        | 5                                         |
| 取值范围      | \[0, 80]                                  |
| 生效范围      | Global                                    |
|是否参与序列化 |是                                          |
